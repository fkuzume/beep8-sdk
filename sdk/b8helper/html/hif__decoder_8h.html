Detailed Description

Module for decoding and managing b8HifEvent events.

This module provides functionality for decoding and managing b8HifEvent events. The CHifDecoder class is used to fetch and manage events from the Hardware Interface (HIF). It stores the events in a map and allows filtering and retrieval of current events.

This module is designed to work with the b8HifEvent structure and related event types.

Features

  • CHifDecoder: Main class for decoding and managing events.
    • GetEvents: Retrieves the current events and updates the internal map.

Usage Example

Here is an example of how to use this module to fetch and process events:

#include "hif_decoder.h"
#include <stdio.h>
int main() {
// Create an instance of the decoder
CHifDecoder decoder;
// Fetch events
const std::map<u8, b8HifEvent>& events = decoder.GetEvents();
// Process events
for (const auto& [id, event] : events) {
printf("Event id: %u, Type: %d\n", id, event.type);
switch(event.type) {
case B8_HIF_EV_TOUCH_START:
printf("Touch Start Event\n");
break;
case B8_HIF_EV_TOUCH_MOVE:
printf("Touch Move Event\n");
break;
case B8_HIF_EV_TOUCH_CANCEL:
printf("Touch Cancel Event\n");
break;
case B8_HIF_EV_TOUCH_END:
printf("Touch End Event\n");
break;
case B8_HIF_EV_MOUSE_DOWN:
printf("Mouse Down Event\n");
break;
case B8_HIF_EV_MOUSE_MOVE:
printf("Mouse Move Event\n");
break;
case B8_HIF_EV_MOUSE_UP:
printf("Mouse Up Event\n");
break;
case B8_HIF_EV_MOUSE_ENTER:
printf("Mouse Enter Event\n");
break;
case B8_HIF_EV_MOUSE_LEAVE:
printf("Mouse Leave Event\n");
break;
case B8_HIF_EV_MOUSE_CLICK:
printf("Mouse Click Event\n");
break;
case B8_HIF_EV_MOUSE_DBLCLICK:
printf("Mouse Double Click Event\n");
break;
case B8_HIF_EV_MOUSE_CONTEXTMENU:
printf("Mouse Context Menu Event\n");
break;
default:
printf("Unknown Event\n");
break;
}
}
return 0;
}
